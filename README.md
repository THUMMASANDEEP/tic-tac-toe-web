# Tic-Tac-Toe Web Application 🎮

A modern, interactive Tic-Tac-Toe game built using HTML, CSS, and JavaScript. This project was developed as **Task-03** for the Web Development internship at **Prodigy InfoTech**.

## 🌟 Features

* **Two Game Modes:** Choose to play against another human (Player vs Player) or test your skills against a basic computer opponent (Player vs AI).
* **Smart Game Logic:** Automatically detects winning combinations and ties, pausing the game appropriately.
* **Modern UI/UX:** Features a sleek dark-blue and vibrant pink color palette with smooth hover and click animations.
* **Responsive Design:** Scales nicely and works perfectly on both desktop and mobile browsers.
* **Zero Dependencies:** Built entirely with plain HTML, CSS, and Vanilla JavaScript—no external libraries required.
